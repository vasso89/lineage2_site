# Lineage 2 Game Server Landing Page

## Description
A responsive landing page for a Lineage 2 game server. Built from a Figma template.

## Features
- Responsive Design - Desktop, Tablet, Mobile
- CSS Grid for cards, Flexbox for navigation
- Semantic HTML5
- Cross-browser Compatibility

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Normalize.css
- Google Fonts (Outfit, Jost)

## Live Demo
https://vasso89.github.io/lineage2_site/

## Installation
1. git clone https://github.com/vasso89/lineage2_site.git
2. Open index.html in browser

## What I Learned
- Pixel-perfect implementation from Figma
- CSS Grid for complex layouts
- Responsive design with multiple breakpoints
- Proper semantic HTML structure
